# 🕹️ Gamers Social - AC1

**Projeto Fullstack desenvolvido em Spring Boot + Frontend HTML/CSS/JS**

## 🚀 Funcionalidades Implementadas
- ✅ CRUD completo de **Jogos** (Criar, Listar, Editar, Deletar)
- ✅ Backend **Spring Boot** com API REST (PostgreSQL)
- ✅ Frontend integrado com tema **Dark Mode** (Steam Style)
- ✅ Testado e funcionando 100%

## 📱 Como testar
1. **Backend:** `cd backend` → `mvn clean spring-boot:run`
2. **Frontend:** Abra `http://localhost:8080`
3. Teste cadastrar/editar jogos na interface

## 📁 Estrutura
